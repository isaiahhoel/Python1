README for PersonType + GameSelect
----------------------------------

PersonType - a program that can be run on its own, however it is meant to be a module for the GameSelect program.  PersonType captures the user's name and age to determine if they are an adult, teenager or child.

GameSelect - a program that imports the PersonType module.  The user type is determined whether or not the user can purchase a game based on the game's rating.  GameSelect allows the user to select and add games to a shopping cart.  When finished, the shopping cart total is displayed.

REQUIREMENTS:	Python 3.9.6
		Both PersonType.py and GameSelect.py must be in the same directory.

USAGE:		python3 GameSelect.py

HOW TO EXIT:	Enter 'X' to properly exit the program.
